XMega For Arduino Project

  • Uses the existing Arduino IDE
  • Adds support code to the existing Arduino environment
  • Patches for 'boards.txt' and 'avrdude.conf'
  • Patches for existing tools, when needed
  • Universal bootloader lets you flash with existing serial protocols
    • 'arduino' (stk500) protocol <= 64k image
    • 'wiring' (stk500v2) protocol > 64k image
