Advanced Capabilities of XMega

  • SRAM/DRAM support (A1 series) as extended 'data' memory
  • Complex Waveform generation (E5 series)
  • Built-in CRC, encryption support (various)
  • Real Time Clock (periodic wakeup from sleep)
  • Software clock control - can reduce clock speed to save power
  • Faster SPI interface (approximately twice the speed of ATMega)
  • Event System (can be used lieu of interrupts)
